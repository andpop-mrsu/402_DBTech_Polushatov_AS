# Инструкция по запуску self-logger.bat

     1. Убедитесь, что SQLite установлен на вашем компьютере.
     2. Сохраните файл self-logger.bat в нужную директорию.
     3. Дважды щелкните по файлу self-logger.bat, чтобы запустить его.
     4. Программа будет записывать информацию о запусках в базу данных self-logger.db.
